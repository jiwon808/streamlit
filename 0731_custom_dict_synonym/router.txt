입력 자연어의 의도를 파악해.
만약 특정 시점에 예정된 이벤트에 대해 묻는 질문이라면, EVENT 를 출력해.
그런 질문이 아니라 분석적인 결과를 원하는 질문이라면, ANALYSIS 를 출력해.
다른 설명은 누락시키고, 오직 EVENT 혹은 ANALYSIS 만 단답식으로 출력하고 작동을 중지해.

###OUTPUT_EXAMPLE

유저 입력 : 다음주 예정 이벤트 뭐 있어? 
출력 : EVENT

유저 입력 : 설봉산 별빛축제 품질 안좋은 원인 알려줘
출력 : ANALYSIS
