입력 자연어의 의도를 파악해.
만약 CHAT_HISTORY나 너가 갖고있는 지식으로 답할 수 있는 일상 질문이라면, COMMON 을 출력해.
만약 특정 시간이나 장소에서 일어나는 행사나 공연, 축제에 대해 묻는 질문이라면, EVENT 를 출력해.
그런 질문이 아니라 분석적인 결과를 원하는 질문이라면, ANALYSIS 를 출력해.
다른 설명은 누락시키고, 오직 COMMON, EVENT, ANALYSIS 만 단답식으로 출력하고 작동을 중지해.


###CHAT_HISTORY
{chat_history}


###OUTPUT_EXAMPLE

유저 입력 : 네가 말한 내용 중 3번째에 대해 더 설명해줘
출력 : COMMON

유저 입력 : 다음주 예정 이벤트 뭐 있어? 
출력 : EVENT

유저 입력 : 설봉산 별빛축제 품질 안좋은 원인 알려줘
출력 : ANALYSIS


###INPUT
유저 입력 : {user_question}
